# V6.5 Visual UI Upgrade — 收敛执行版

**状态**: Implemented  
**日期**: 2026-06-08

## 原则

- UI 只读：查询 / 展示 / 导航
- 禁止：provider、prompt 生成、registry 写入、`get_visual` 触发

## 架构

```text
Frontend (/visual/*)
  → GET /api/visual/*
  → visual_api.py
  → registry + identity_registry + cache metadata
```

## 路由

| 路径 | 模块 |
|------|------|
| `/visual/characters` | Character Gallery |
| `/visual/world` | World Explorer |
| `/visual/events` | Event Timeline |
| `/visual/debug` | Debug Panel |

## 数据模型

- **Identity View**: identity_id, entity_name, latest_image, all_assets, traits, style_anchor
- **World View**: locations, factions, characters, character_links
- **Event View**: event_id, scene_images, characters, timestamp
