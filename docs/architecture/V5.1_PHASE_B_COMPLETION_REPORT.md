## 数据结构

**路径**: `data/relationship_memory.json`

```json
{
  "version": 1,
  "edges": {
    "主角->长公主": [
      {
        "turn": 12,
        "actor": "主角",
        "target": "长公主",
        "type": "rescue",
        "summary": "救下刺客事件",
        "trust_delta": 5,
        "affection_delta": 8,
        "respect_delta": 0,
        "hostility_delta": 0
      }
    ]
  }
}
```

**边键**: `actor->target`（有向）

**事件类型**（固定枚举，禁止 AI 发明）:  
`first_meeting`, `support`, `betrayal`, `rescue`, `protect`, `confession`, `romance`, `argument`, `conflict`, `gift`, `cooperation`, `separation`, `reunion`

**自动记录触发**:
- 任一维度 delta ≥ 5（0–100 刻度），或
- `relation_type` 变化

规则生成 `type` / `summary`，不调用 AI。

## Director / Brain 接入

| 消费者 | API | 行为 |
|---|---|---|
| Character Brain | `read_api_for_brain` + `format_brain_memory` | 图谱状态 + 最近关系事件 |
| Objective | `read_api_for_objective_text` | 目标进度 + 最近事件 |
| Plot Director | `read_api_for_plot` | 关系张力与修复建议（只读） |
| Event Director | `build_relationship_event_candidates` | 规则候选（非强制） |

Plot Director **不修改**关系图。

## Save / Reset

| 场景 | 行为 |
|---|---|
| `RuntimeState` | 新增 `relationship_memory` 字段，`commit_runtime` 原子写入 |
| `save_manager` | 存档 JSON 含 `relationship_memory` |
| `world_init.json` | 新故事写入 `relationship_graph` + `relationship_memory` |
| `/reset` | 从 `world_init` 恢复两者（`commit_bundle` 同事务） |
| `reset_user_data.py` | 清理 `relationship_memory.json` |
