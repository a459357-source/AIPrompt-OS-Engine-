## V6.5 UI（World Viewer 升级）

- 路由：`/visual/characters` | `/visual/world` | `/visual/events` | `/visual/debug`
- Identity View：`all_assets` + `latest_image` + traits（按 identity_id 聚合）
- World View：locations / factions / characters / character_links（只读）
- Event View：时间排序、可展开、scene_images + 参与角色
- Debug：`registry_id`、`cache_hit`、`prompt_hash`、`identity_id`
- 只读 API，不触达 provider / runtime 生成
