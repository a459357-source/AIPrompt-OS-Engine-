## V6.1 Visual Identity Layer

```text
entity_id → visual_identity → prompt → image
```

| Module | 文件 |
|--------|------|
| VisualIdentity | `engine/visual/visual_identity.py` |
| Identity Registry | `engine/visual/identity_registry.py` |
| Identity Prompt | `engine/visual/identity_prompt_builder.py` |
| Migration | `visual_object.py` / `visual_runtime.py`（V6 API 不变） |

### 强规则

- identity 权重大于 context
- 角色：context 仅可修饰 pose/background/lighting
- 同一 identity → 同一 idempotency（角色按 identity_id 锁定）
- prompt 可变修饰，identity 不可变
