## 流程（Phase B）

```text
get_or_request_*()
    ↓
Registry[asset_id] 有效？ → 返回
    ↓
Registry 同 scope 同 prompt_hash？ → 复用 image_path
    ↓
Cache 文件存在？ → 写 Registry 返回
    ↓
Provider.generate_*() → Cache 写盘 → Registry 记录
```

**原则**：同 Prompt 不重复请求 Provider。
