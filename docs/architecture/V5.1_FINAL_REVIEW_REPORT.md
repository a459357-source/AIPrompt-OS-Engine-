## 观察项（不阻断 V5.2）

1. **Triangle 触发率低** — 模拟世界 NPC↔NPC 边不足；建议 V5.2 用更完整 world_pack 回归。
2. **Bond 分布偏高** — 测试初始关系偏高导致大量 Bond=4；真实新故事更低。
3. **Prompt 块重叠** — `CHARACTER_BRAIN` 关系块与 `MEMORY`/`DYNAMICS` 占位符部分重复；worst-case 仍 < 1500 tokens。
4. **Memory 稳定后停录** — delta < 5 且无 bond 变化时不追加事件（设计行为）。
