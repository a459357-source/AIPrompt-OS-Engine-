# V6.6 Narrative Entry Layer

**状态**: Implemented  
**日期**: 2026-06-08

## 定位

在 V6.5 世界浏览器之上，增加**叙事入口层**：路由 + 展示 + 状态切换。  
**不触发生成**：无 `get_visual`、无 provider、无 LLM。

## 架构

```text
UI (Explore / Narrative toggle)
  → GET/POST /api/narrative/*
  → narrative_router + narrative_entry
  → registry / identity / narrative_routes.json
```

## API

| 端点 | 职责 |
|------|------|
| `GET /api/narrative/node/{event_id}` | 叙事节点视图 |
| `GET /api/narrative/choices/{event_id}` | 选项列表 |
| `POST /api/narrative/route` | choice → next_event_id |
| `POST /api/narrative/enter/event\|character\|location` | 三类入口 |
| `POST /api/narrative/mode` | explore ↔ narrative |

## 数据

- `data/narrative_routes.json` — 节点 context + choices 映射
- `data/narrative_state.json` — 当前模式与节点（与 explore 隔离）

## UI 路由

- `/visual/narrative` — 入口中心
- `/visual/narrative/node/:eventId` — 节点视图 + 选择
