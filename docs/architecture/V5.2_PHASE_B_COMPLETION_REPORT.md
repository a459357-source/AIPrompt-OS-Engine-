## 交付摘要

将 Event Director 从 **推荐器** 升级为 **调度器**：

- 引入 `current_event`（ACTIVE 剧情节点）
- 生命周期：`PENDING → ACTIVE → RESOLVED/FAILED → COOLDOWN`
- 回合前 `prepare_director_prompt` 提升/维持 ACTIVE 事件
- 回合后 `advance_director_after_turn` 检测收束或超时
- 冷却仅在事件 **完成生命周期** 后写入 `event_history`

Story / Adult **共用** `director_state.py` + `director_runtime.py`，无模式分叉。

## 数据模型

### `data/director_state.json`

```json
{
  "version": 2,
  "current_event": { "event_id": "midnight_talk", "state": "ACTIVE", ... },
  "pending": [],
  "lifecycle": []
}
```

### 生命周期

```text
评分候选 (build_director_plans)
    ↓
TOP1 → ACTIVE (current_event)
TOP2-3 → PENDING (queue)
    ↓
Prompt：须推进 ACTIVE 节点
    ↓
回合结束：story 信号 → RESOLVED
         或超时 → FAILED
    ↓
COOLDOWN → event_history（冷却计时）
```
