## 规则摘要

### Momentum
- 正向互动：`momentum += min(8, positive_delta * 0.6)`，上限 30
- 无互动回合：`momentum -= 2`
- 正向 delta 乘数：`1 + momentum * 0.015`（最高约 +30%）

### Decay
- `inactive_turns > 20`：每回合 trust -Δ、affection -0.8Δ（Δ≤2）

### Bond Level（0–5）
由 trust/affection/respect/dependence/hostility 推导：陌生→认识→熟悉→信任→亲密→羁绊

### Conflict Level（0–5）
由 hostility/trust/respect 推导：无→摩擦→对立→敌视→仇恨→死敌

### Influence（一层）
- 玩家亲近 A，第三方对 A 敌意 ≥40 → 第三方对玩家敌意上升
- A 好感 B、B 敌意 C、玩家亲近 A → 玩家对 C 轻微负面
- 系数：`RELATIONSHIP_INFLUENCE_COEFF = 0.2`

### Triangle Detection
识别 support_rival / ally_enemy_spill / romance_triangle，写入 `triangles` 供 Plot Director

### Event Resolver
Bond level 上升时自动写入 `relationship_memory.json`（规则 summary，无 AI）
