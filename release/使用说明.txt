PromptOS 发布包使用说明
====================

【给别人用】
1. 只发送 release\PromptOS-win64-v{版本}.zip（不要附带 data 文件夹）
   例：PromptOS-win64-v2.0.1.zip
2. 对方解压后双击 PromptOS.exe
3. 首次打开会弹出窗口，请填写各自的 DeepSeek API Key
   （也可在「设置」页填写；Key 仅保存在本机 data\apikey.json）
4. 通过「新故事」开始游戏

【重要】
- 发布包内不含任何 API Key
- exe 不会读取打包者电脑上的 DEEPSEEK_API_KEY 环境变量
- 若在本机测试 zip，请先删除系统环境变量 DEEPSEEK_API_KEY，否则会误用你的 Key

【重新打包（确保无个人数据）】
  cd prompt-os-engine

  发布类型          版本示例    命令
  ─────────────────────────────────────────────────────────
  修 bug / 小更新   2.0.1       python build_release.py
  新功能（兼容）    2.1.0       python build_release.py --version 2.1.0
  大改版（不兼容）  3.0.0       python build_release.py --version 3.0.0
  重打同版本 zip    2.0.0       python build_release.py --no-bump

  完整说明见仓库 RELEASE_VERSIONING.md 与 .cursor/rules/release-build.mdc

【日志】
- 运行日志: PromptOS\data\app.log
- 错误日志: PromptOS\data\error.log
- 用「启动 PromptOS.bat」或双击 PromptOS.exe 启动后自动生成

打包前请勿在 dist\PromptOS 目录运行过程序后再手动打 zip。
