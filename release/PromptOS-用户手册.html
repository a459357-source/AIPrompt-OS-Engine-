1. 产品简介

🏰
世界构建
角色 · 势力 · 关系
🎮
回合推进
AI 叙事 + 分支选项
📊
世界状态
关系网 · 分支图 · 时间线

PromptOS（Prompt OS Galgame Runtime）是一款面向互动小说 / Galgame 式叙事的创作与游玩工具。你设定世界观与角色，由 DeepSeek 大模型每回合生成小说正文与后续选项，引擎自动维护记忆、关系指标与剧情图。

v2 主要能力

  • 分层记忆：长对话下自动压缩历史，保留关键情节
  • 世界状态 V2：仪表盘展示势力、关系网、分支图、时间线
  • 生成参数联动：目标字数与 Token 上限自动匹配
  • 成人模式（可选）：顶栏切换，世界生成与叙事风格跟随
  • 一键 AI 辅助：世界观、角色、势力、规则字段可 AI 生成

2. 安装与启动

Windows 发布包（推荐给普通用户）

  1. 解压 PromptOS-win64-v2.x.x.zip（勿附带他人的 data/ 文件夹）
  2. 双击 PromptOS.exe，或运行同目录下的 启动 PromptOS.bat
  3. 浏览器会自动打开 http://127.0.0.1:8000（内置前端，单端口）
  4. 首次使用按弹窗填写 DeepSeek API Key（见下一章）
  5. 进入「世界」页创建故事并开始模拟
注意：发布包不含任何 API Key；exe 不会读取打包者电脑上的 DEEPSEEK_API_KEY 环境变量。在本机测试 zip 前，建议临时删除该环境变量，避免误用他人 Key。

开发者模式

# 后端
pip install -r requirements.txt
python engine/run.py --mode web    # → http://localhost:8000

# 前端（另开终端）
cd frontend && npm install && npm run dev   # → http://localhost:5173
模式地址说明
exe 发布包127.0.0.1:8000前后端一体
开发后端 8000 + 前端 5173前端代理到后端 API

3. 配置 API Key

图 3-1　首次启动时的 API Key 引导弹窗

获取 Key

在 DeepSeek 开放平台 注册并创建 API Key（格式 sk-...）。

填写方式（二选一）

  • 首次弹窗：启动后自动出现，保存至本机 data/apikey.json
  • 设置 → API：可随时修改、清除 Key；清除后弹窗会再次出现
  • 环境变量（仅开发）：DEEPSEEK_API_KEY=sk-...
隐私：Key 仅保存在本机，不会随 zip 发布包分发。请勿将含 Key 的 data/ 文件夹发给他人。

5. 世界 / 新故事

图 5-1　世界构建页：左侧分节导航 + 右侧表单与关系图

推荐流程

  1. 核心设定：填写标题（≤20 字）、类型标签、开局地点、世界观、主线目标
  2. 势力：添加组织/阵营，可 AI 生成名称与关系
  3. 角色：至少 1 名角色；可设主角色、外貌、性格标签、背景、秘密、能力
  4. 关系：配置关系阶段轴（如「陌生→信赖→羁绊」）与初始好感
  5. 关键物品（可选）：剧情道具与归属
  6. 点击 「开始故事」 — 会覆盖当前对局进度，操作不可撤销

AI 辅助按钮

  • ✨ 一键生成：用 AI 填充整表（会覆盖当前全部设定，需确认）
  • 各字段旁的 AI 按钮：单独生成世界观片段、角色、势力等
  • 表单内容会自动本地草稿保存，刷新页面可恢复
开始新故事将初始化全新对局，当前进度、剧情记录与相关存档会被覆盖。

6. 模拟（游戏）

图 6-1　模拟页：预设选项（A–D）或底部「自定义输入」均可推进回合

基本操作

  • 阅读 AI 生成的 Markdown 正文（支持打字机效果，可在设置 → 界面中关闭动画）
  • 推进回合（二选一）
    • 点击 AI 给出的 A / B / C / D 预设选项（选项数量可在快捷设置中设为 3–5 个）
    • 或点击选项下方的 「✏️ 自定义输入…」，在输入框中写下你想做的行动，按 Enter 或点「确定」提交
  • 自定义内容与预设选项等效：引擎会把你的文字作为「玩家选择」交给 AI 续写，历史中会标记为「✏️ 自定义」
  • 若当前没有可用选项（例如生成延迟），也可直接使用自定义输入继续推进
  • 左侧面板显示当前角色关系指标（信任、好感、敌意等）
  • 顶栏显示当前回合数、剧情状态机（SETUP → BUILD → TENSION → CLIMAX → COOLDOWN）、场景
自定义输入提示：尽量写清楚具体行动（如「悄悄跟踪她到后院」），比单字描述更容易得到连贯剧情。按 Esc 可取消输入框。若内容与成人向相关，非成人模式下可能出现开启成人模式的提示横幅。

⚡ 快捷设置（游戏页）

展开「⚡ 快捷设置」可调整下一回合生效的生成参数：

参数说明
目标字数每轮期望叙事长度；变更后 最大 Token 自动联动
最大 Token单次 API 输出上限（含选项与状态 JSON）
温度 / Top P创意度；推荐温度 0.8–1.0
选项数量每轮 3–5 个选项
叙事视角 / 文风第一/第三人称、细腻/快节奏等
重复检测严格模式下相似度过高会重试
沉浸阅读隐藏侧栏与快捷设置，专注正文

模型选择、流式输出、上下文上限、自动压缩开关在 设置 → API 中配置。

输出过短？若提示 token 不足，请展开快捷设置提高「目标字数」或「最大 Token」。

9. 设置

📖 阅读

字体大小、行高、正文字体、最大宽度、背景色调（深色/护眼/灰）、段落间距；含实时预览。

💾 数据

自动保存间隔、最大存档槽位、导出格式、自动导出频率；手动存档槽；重置游戏进度；Obsidian 导出。

🎨 界面

动画开关、侧栏默认状态、角色面板位置、语言（中/英/日部分支持）。

🔑 API

DeepSeek Key、模型（V4-Flash / V4-Pro）、流式输出、上下文消息上限、自动压缩。

成人模式（可选）

顶栏可切换成人主题与内容权重（故事 / 恋爱 / 成人）。开启后导航文案与视觉主题会变化，世界一键生成与叙事风格会跟随成人配置。

12. 常见问题

❓ 启动后浏览器空白或无法连接

确认 PromptOS.exe 窗口未关闭；检查 8000 端口是否被占用。可先运行「停止.bat」再重启。

❓ API 报错 / 401 / 余额不足

检查 Key 是否正确、DeepSeek 账户是否有余额。在设置 → API 重新保存 Key。

❓ 生成内容过短或被截断

在游戏页 ⚡ 快捷设置中提高「目标字数」和「最大 Token」；必要时在设置 → API 关闭或调整自动压缩。

❓ 历史太长导致变慢或超上下文

开启「自动压缩」并适当降低「上下文消息上限」；压缩阈值可在游戏快捷设置中预览。

❓ 如何完全重新开始

在世界页点击「开始故事」（重建对局），或在设置 → 数据 →「重置游戏进度」（保留 world_pack）。

❓ 关闭程序

直接关闭 PromptOS.exe 的黑色控制台窗口即可停止服务。开发模式下设置页提供「关闭服务」。
